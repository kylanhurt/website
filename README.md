# Wharfkit Website

The website for https://wharfkit.com/

## Developing

Clone this repo `git clone https://github.com/wharfkit/website`

Run `make dev`

### Subset fonts

`glyphhanger --subset=*.ttf --formats=woff2 --US_ASCII`
